# Portfolio Starter (Vite)

This is a Vite + React version of the portfolio starter template.

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173 to view the app.

## Environment

Create a `.env` file in this folder and set:

```
VITE_USER_ID=2
VITE_BACKEND_API_URL=https://aetrix-backend-git-master-ashintvs-projects.vercel.app/
```

## Routes

- `/` portfolio page
- `/components` component showcase
